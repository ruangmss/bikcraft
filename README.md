# Bikcraft

Projeto desenvolvido para fins de estudo durante o curso de **HTML e CSS** da [Origamid Cursos](https://www.origamid.com/).  
O site apresenta a marca fictícia **Bikcraft**, especializada em bicicletas elétricas e planos de seguro, e foi construído com foco em **HTML e CSS**.

---

## Sobre o projeto

O Bikcraft é um site institucional fictício que simula uma empresa de bicicletas elétricas.  
Ele contém páginas de apresentação dos produtos, planos de seguro, formulário de orçamento e informações de contato.

O projeto está publicado em GitHub Pages e pode ser acessado aqui:
[Visualizar o site](https://ruangmss.github.io/bikcraft/index.html)

---

## Tecnologias utilizadas

- **HTML5** → Estrutura semântica das páginas
- **CSS3** → Estilização, responsividade e animações
- **JavaScript** → Interatividade e manipulação do DOM

---

## Funcionalidades

- Navegação entre páginas (Bicicletas, Seguros, Contato, Orçamento)
- Formulário de orçamento com seleção dinâmica de produtos ou planos de seguro
- Exibição condicional de blocos conforme a escolha do usuário
- Layout responsivo para diferentes dispositivos

---

## Aprendizados

- Estruturação de sites com HTML semântico
- Criação de layouts responsivos com CSS
- Uso de variáveis, pseudo-classes e boas práticas de estilização
- Manipulação de elementos com JavaScript para criar interatividade

---

## Observação

- Este projeto é apenas para fins educacionais e não possui finalidade comercial.
